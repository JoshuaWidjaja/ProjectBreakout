DOCUMENTATION
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
BallV3.py - Contains the GameBall class and all of it's attributes/helper functions. Used to allow the GameBall to move in different directions after each bounce and adjust
the speed of the ball while playing.

BrickV3.py - Contains the Brick class and all of it's attributes. Used to ease the process of creating bricks for the player to break in the game.

PaddleV3.py - Contains the Paddle class and all of it's helper functions. Helper functions allow player to move the paddle left and right as well as start the game.

ProjBreakoutV3.py - Main file. To launch the game, launch from this file. Contains game logic, helper functions, and the primary game loop that allow the player to play.
Game ends when either all lives are lost or all bricks are destroyed, in which case the program will close.
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
CONTROLS
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Left Arrow - Move paddle left

Right Arrow - Move paddle right

Spacebar - Launches ball from paddle at the start of the game or whenever a life is lost.
